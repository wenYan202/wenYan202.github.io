IoT- Smart Trash can Management System

17-01-2020

Introduction

The system is based on Arduino and ultrasonic sensors to monitor the fullness of the trash cans and to send the collected data to Clould using Rasberry PI with WIFI. The cleaners can know the situation of each trash can at any time through android application.

System Architeture
This the whole architecture of the whole system the sensor node and the trash can compose the smart can, Arduino sent the measured data to the gateway through Zig Bee communication. Then the gateway sends the data to a remote Azure cloud database and the data can also be stored here. Then user can use Android app to access the data.

Material

  • A. Arduino Uno
    It’s a single-board microcontroller, designed to make the application of interactive objects or environments more accessible. Sense the environment by receiving input from variety of sensors.
  • B.Ultrasonic Sensor
    This is the working principle of the ultrasonic sensor which is in the top of trash can. it used for distance measuring which uses reflection mechanism. As soon as there is a signal to sensor, it start to send waves and when it hit an object, it will reflect, and then the echo pin can receive it. That's how it calculates the distance.
  • C. Rasberry PI
    Raspberry Pi is a low cost, credit card sized, computer that performs various applications. Some of its main features include 1GB of RAM, 4 USB Ports, General Purpose Input Output pins, Linux support. These features give programmers a wide range for diverse applications. It is used as gateway in the project.
  • D. Zig bee module
    Zig Bee is an IEEE 802.15.4-based specification for a suite of high-level communication protocols used to create personal area networks with small, low-power digital radios, such as for home automation, medical device data collection, and other low-power low-bandwidth needs, designed for small scale projects which need wireless connection.

Implementation
The framework of the system can be divided into 4 stages:

    1. The Smart Bin: sensor node is mounted on the bin that composed the smart bin. Ultrasonic sensors are used for level measurement of garbage in the bin. Arduino sent the measured data to the gateway through Zig Bee communication.
    1. Gateway: the status of trash cans is collected to IoT gateway through Zigbee receiver module. Then the gateway sends the data and also stores to a remote Azure database.
    1. Cloud Database: a database created in Azure cloud-platform to store information.
    1. User Interface: a android-based graphic user interface is designed to help user see the fullness of each trash can at any time.


User Interface
This is the user interface of the system, first user need to login the system, and then choose layer and the trash cans distribution of this layer can be seen. when user clicks the one can, it will access database to get the current height of the garbage. Then the fullness of the trash can will be displayed.
