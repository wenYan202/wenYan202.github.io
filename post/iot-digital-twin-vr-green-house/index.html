IoT Digital Twin-VR Green House

16-01-2020

Conception
A digital twin is the concept of building the connection between digital world and physical world. In the project, we designed a wireless sensor network that provides real-time monitoring of light, temperature, humidity of the greenhouse.

System Architecture
This is the architecture of the system, let’s say that there is a greenhouse in the distance where you cannot monitor it in person, and there are some different sensors installed in it, like heat sensor, light sensor and humidity sensor, and the data collected from sensor is sent to the IoT gateway which here is Raspberry PI. then the Raspberry PI upload the information to remote server. The server runs on computer can also store data in database.

To enhance the user's sensory perception, we build a 3D greenhouse by Unity, then the user can interact with the 3D scene and use the VR device to view the environmental information inside of the greenhouse.
And also we made a very simple user interface to let user control the light which is in the actual greenhouse, like switch the light on or off. and at the same time the scene you view with VR is also changed.

3D Scene
Here are some 3D greenhouse scene we have build on Unity. The first one is the inside, here you can see the light is on now. Another one is the outside of greenhouse.

The Sensor Node:
Arduino works with different sensors to collect data/ control the switch related to the command.
IoT Gateway:
Raspberry PI is used as a gateway to collect data from Arduino and upload the information to remote server. Download commands from Server and transmits to Arduino.
Server:
Server runs on a tablet computer. It stores data from IoT gateway in database.
User Interface:
Client provides users with a 3D greenhouse scenes to allow them manage and view information.
